How to use this website template:

1. Double-click index.html to preview the website.
2. Edit index.html to change the text.
3. Edit style.css to change colours, fonts, spacing, and layout.
4. Edit script.js to change travel map locations and descriptions.
5. To publish:
   - easiest: upload to Netlify Drop
   - or use GitHub Pages
   - or copy the text/design into Framer or Webflow

Suggested next edits:
- Replace your.email@example.com with your real email.
- Add your own essays as separate HTML pages.
- Replace the abstract map with a real SVG/world map later.
- Add photos from travel, hikes, libraries, and small discoveries.
